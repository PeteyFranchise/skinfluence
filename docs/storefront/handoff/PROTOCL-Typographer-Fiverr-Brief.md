# PROTOCL Typographer Brief

## Project Context

We are rebranding an in-progress luxury skincare website and packaging system from `Skinfluence` to `PROTOCL`.

The website is already partly built, so this is **not** a blank-slate identity project. We need a typographer/brand designer who can refine and elevate an existing direction into a final, polished logo system and mini brand kit that fits the site and packaging we are already developing.

## Primary Goal

Create a final, improved brand wordmark and supporting mini brand kit for `PROTOCL` that feels:

- luxury
- sharp
- clinical-editorial
- premium enough for website, packaging, and unboxing content

This should feel at home next to brands like **ALLIES OF SKIN**, while still becoming clearly ownable to `PROTOCL`.

## Important Project Constraint

This is an **evolution/refinement of an existing direction**, not a full reinvention from zero.

The website already has a live design language, and the identity should fit inside that system rather than fight it.

## Brand Name

- Official name: `PROTOCL`
- Default presentation should be **all caps**
- Some creative leeway is okay, but all-caps should remain the anchor

## Existing Typography System To Respect

Please design with the current website typography system in mind:

- Wordmark/reference serif direction: `Bodoni Moda`
- Supporting purchased website font: `Graphik Compact`

Important:

- We want at least **one concept** built from a Bodoni-like serif base
- Packaging/supporting descriptions should harmonize with the `Graphik Compact` direction

## Existing Symbol Direction

We currently like a direction where:

- the second `O` in `PROTOCL` is replaced by a symbol
- that symbol is a **square dot inside brackets**
- the same symbol is also a strong direction for the favicon/app icon

Requirement:

- in at least **one concept**, please keep that symbol integrated into the main wordmark

## Deliverables Required

Please provide:

1. Primary `PROTOCL` wordmark
2. Header-optimized version for a slim website nav/header
3. Standalone symbol/monogram
4. Favicon/app icon version
5. Social-media-ready logo/avatar treatments
6. Black-on-white version
7. White-on-black version
8. Packaging lockup examples
9. Embossed/foil-friendly variation
10. Mini brand sheet / mini brand kit
11. Supporting icon elements
12. Supporting pattern elements

## Final Files Required

Please include:

1. `AI`
2. `SVG`
3. `EPS`
4. `PDF`
5. Transparent `PNG` exports

## Mini Brand Kit Requirements

Please include a slightly fuller mini brand kit suitable for **web + packaging** use, including:

- logo usage guidance
- clear space / spacing rules
- minimum size guidance
- black-and-white-first usage
- a small core color palette
- icon/pattern usage guidance
- packaging/web application guidance

## Color Direction

Please keep the system:

- black-and-white first

But also define:

- a small core color palette

This palette can either:

- build from tones already present in the project
- or propose a fresh but compatible palette

## Packaging Constraints

The identity must work well for:

- matte-black cartons
- small bottle labels
- unboxing content
- foil applications
- embossing/debossing

Please include packaging-aware guidance for how the logo should behave in these use cases.

## Supporting Icons + Patterns

Please create:

- icon elements
- pattern elements

These should stay **closer to the current system**, rather than reinventing the brand into something unrelated.

## Concept Direction

Please present:

- **3 initial concepts**

Guidance:

- at least 1 concept should remain close to the current Bodoni-like direction
- the others can be neighboring elevated explorations

We are open to:

- a fully custom-drawn wordmark
- or a refined/customized wordmark built from a Bodoni-like base

But at least one direction should clearly honor the existing Bodoni-based lane.

## What Success Looks Like

We do **not** dislike the current temporary direction.

We simply want:

- a final, improved version
- more polish
- stronger spacing/proportion
- a more recognizable and ownable result

The final mark should feel:

- recognizable
- premium
- custom enough to feel brand-owned

## Mockups Requested

If possible, please also show the logo applied in context using:

- packaging mockups
- website/header mockups

If you can provide **AI-generated packaging mockups** we may be able to use on the website, that would be especially helpful.

## Timeline

- Revisions expected: **2 rounds**

- anything that loses the premium sharpness of the current system
