# Skinfluence Packaging Next Steps

## Purpose
This file is a reminder of what still needs to happen to fully finalize the Skinfluence packaging system after the current website-image phase.

Right now, the front-facing concept direction is strong enough to support website imagery. The remaining steps below are for turning that concept into a full, production-ready packaging system later.

## Current Concept Direction

- slim cylindrical airless treatment pump bottle
- matching tall rectangular carton
- matte black / near-black finish
- premium clinical-luxury positioning
- restrained, editorial hierarchy
- bottle carries more detail than carton
- carton stays cleaner and more brand-forward

## What Still Needs To Be Finalized Later

### 1. Lock the Front-of-Pack System
Revisit the current Figma front label and confirm:

- final descriptor wording
- final actives list
- final icon set
- final logo treatment
- final carton/bottle hierarchy

This should only happen once the chemist or formulation partner confirms what can actually appear on pack.

Also confirm:

- whether the Midjourney-inspired icon style remains the approved packaging icon direction
- when that icon direction should be redrawn as final vector artwork

### 2. Design Carton Side Panels
The carton still needs structured side-panel design for:

- treatment explanation
- ingredient or active story
- usage instructions
- warnings or caution language
- bottle size / net contents
- provider or prescription positioning if relevant

Goal:
- structured, premium, grid-based side panel
- no crowded generic information blocks

### 3. Design Carton Back Panel
The back panel still needs:

- longer product explanation
- usage guidance
- storage guidance if needed
- supporting claims hierarchy
- contact / brand / compliance information

This should feel premium and controlled, not overfilled.

### 4. Confirm Formula-Based Copy
The current actives and descriptor language are placeholders and must later be aligned with:

- the real chemist/formulary
- regulatory constraints
- claim substantiation
- final dosage / use pattern

This affects:

- front label
- side/back panels
- website PDP copy
- launch marketing language

### 5. Finalize Print and Finish Specifications
Still needed:

- soft-touch carton finish decision
- matte black material specification
- optional foil / metallic accent decision
- optional spot UV / spot gloss decision
- print tolerance review for small text and icon details

This work should be done with the packaging printer or production partner.

### 6. Create Production-Ready Artwork
Later, convert the concept-stage design into:

- dieline-ready carton artwork
- production-ready bottle label artwork
- print-ready logo files
- production-ready vector icon set based on the approved packaging icon direction
- outlined or production-safe typography where needed
- final packaged asset export set

### 7. Build a Multi-SKU System
Once the first product is finalized, test the system across additional SKUs so the brand holds together across:

- different active combinations
- different use cases
- potential AM/PM distinctions
- future expansion products

The goal is a packaging system, not just a single good-looking bottle.

### 8. Create the Website Image Set
Separate from packaging production, we still need a polished website image set based on the approved front-facing packaging direction.

Suggested website image set:

- bottle + carton hero image
- single bottle isolated packshot
- angled editorial product shot
- optional routine or regimen grouping shot

## Recommended Order When We Return

1. confirm front label with real formula inputs
2. finalize carton front
3. design side/back panels
4. align with printer on finishes
5. create production-ready artwork
6. extend to additional SKUs

## Reminder
For now, we are pausing the full packaging system because the immediate need is website imagery. This file is here so the packaging system can be resumed later without losing the logic of what remains.
