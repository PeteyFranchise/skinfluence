# Skinfluence Image Slot Guide

This guide maps the current Shopify theme image surfaces to the pages where they appear, what each slot is meant to communicate, and what kind of visual should be uploaded there.

## Current High-Value Image Slots

### 1. Homepage hero
- Section: `Skinfluence Hero`
- Page: `Home page`
- Field: `Hero image`
- Alt text field: `Hero image alt text`

Recommended direction:
- editorial portrait or upper-body crop
- calm, premium, refined
- soft natural or diffused studio light
- neutral wardrobe and background

Suggested alt text pattern:
- `Woman with calm, confident expression in soft natural light`

### 2. Homepage side-by-side block
- Section: `Side-by-Side Content`
- Page: `Home page`
- Field: `Image`
- Alt text field: `Image alt text`

Recommended direction:
- close editorial still life
- product-adjacent or skin-quality-oriented image
- strong negative space and clean composition
- should support the “premium anti-aging care” narrative rather than read like generic stock

Suggested alt text pattern:
- `Editorial skincare still life in warm neutral light`

### 3. Start Evaluation hero
- Section: `Skinfluence Hero`
- Page: `Start Evaluation`
- Field: `Hero image`
- Alt text field: `Hero image alt text`

Recommended direction:
- process-calming still life
- desk, notebook, skincare, or hands in a quiet setting
- organized, reassuring, and low-friction feeling

Suggested alt text pattern:
- `Calm intake-inspired still life with skincare and notebook`

### 4. Support hero
- Section: `Skinfluence Hero`
- Page: `Support`
- Field: `Hero image`
- Alt text field: `Hero image alt text`

Recommended direction:
- warm human support context
- calm desk scene or soft communication-oriented visual
- should feel organized and reassuring, not corporate

Suggested alt text pattern:
- `Warm support-oriented workspace in soft natural light`

### 5. Support contact visual
- Section: `Support Contact`
- Page: `Support`
- Field: `Support visual`

Recommended direction:
- abstract or semi-abstract branded still life
- should support the form card without competing with the hero
- more system-oriented and structured than emotional

Suggested alt text approach:
- leave decorative if the image is purely atmospheric
- if it carries meaning, match the content precisely

## Pages That Already Support Hero Images

The `Skinfluence Hero` section supports image upload and alt text on:
- homepage
- start-evaluation
- support
- pricing
- faq
- about
- how-it-works
- treatment
- legal pages

Not every page needs a real image immediately. Prioritize:
1. homepage
2. start-evaluation
3. support
4. pricing

## Visual Priority Order

If bandwidth is limited, upload in this order:
1. homepage hero
2. homepage side-by-side block
3. start-evaluation hero
4. support hero
5. support contact visual

## Usage Notes

- Avoid using the same image in multiple slots.
- Keep imagery consistent with the Skinfluence art direction:
  - premium
  - clinically grounded
  - calm
  - editorial
- Prefer real photography over the fallback SVG artwork wherever possible.
- The fallback SVGs are safety nets, not final design assets.
