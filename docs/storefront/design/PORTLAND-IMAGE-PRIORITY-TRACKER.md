# Portland Image Priority Tracker

This document tracks the next image pass for the Portland storefront.

It exists to answer one practical question:

**Which image slots should be filled first, and what should go in them?**

## Priority order

### Tier 1

Highest-visibility, highest-impact slots:

1. Treatment hero
2. Treatment `What this is` split image
3. Pricing hero
4. Start Evaluation hero
5. Start Evaluation `What happens here` split image

### Tier 2

Important but slightly less urgent:

6. Pricing `What’s included split` image
7. How It Works hero
8. How It Works `Process overview` split image
9. About hero
10. About `Origin story` split image

### Tier 3

Helpful polish after the high-impact slots are filled:

11. Support hero
12. Support `Routing overview` split image
13. FAQ hero
14. FAQ `FAQ intro` split image

## Working rule

Do not try to fill every image slot at once.

Use this sequence instead:

1. complete Tier 1
2. evaluate the visual improvement
3. then move to Tier 2
4. leave Tier 3 for polish if needed

## Current recommendation

If only one page gets images first:

- start with `Treatment`

If two pages get images first:

- `Treatment`
- `Pricing`

If three pages get images first:

- `Treatment`
- `Pricing`
- `Start Evaluation`
