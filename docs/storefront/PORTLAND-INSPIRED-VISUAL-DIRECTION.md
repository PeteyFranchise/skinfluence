# Skinfluence Portland-Inspired Visual Direction

This document captures how the Shopify theme `Portland` should influence the Skinfluence storefront without requiring a theme switch or a rebuild.

Portland is not the target implementation. It is the visual reference.

## Why Portland Fits Skinfluence

Portland is a strong reference because it leans:
- editorial instead of generic ecommerce
- content-forward instead of catalog-heavy
- minimal but not sterile
- premium without looking flashy

That makes it a better reference for Skinfluence than standard product-grid-first Shopify themes.

Skinfluence needs to feel:
- premium
- clinically grounded
- calm
- feminine without beauty clichés
- editorial rather than generic DTC

Portland supports that direction because it emphasizes:
- clean spacing
- storytelling blocks
- restrained visual hierarchy
- full-width media moments
- content that feels designed, not merely arranged

## Core Direction to Borrow

### 1. Editorial pacing
The site should feel like a sequence of designed moments, not a repeated stack of identical sections.

Skinfluence should borrow:
- stronger contrast between sections
- more deliberate pacing between dense and light moments
- clear visual breathing room

### 2. Fewer “default ecommerce” signals
Portland feels less like a store template and more like a designed brand experience.

Skinfluence should borrow:
- less visual sameness across page heroes
- more considered use of imagery
- cleaner balance between copy and media

### 3. Storytelling over transaction-first design
Skinfluence should not feel like a hard-sell skincare storefront.

Portland reinforces:
- narrative flow
- visual trust
- calm conversion

That is useful for Skinfluence because the brand needs to lead users into:
- trust
- explanation
- evaluation

not a rapid add-to-cart flow.

### 4. Premium restraint
Portland’s value is not maximalism. It uses restraint well.

Skinfluence should borrow:
- fewer loud visual effects
- more material contrast
- careful use of banded sections
- stronger image choice rather than more decorative design

## What Skinfluence Should Not Copy

Skinfluence should not inherit Portland literally.

Avoid copying:
- fashion-brand styling cues that feel too lifestyle-forward
- overly soft or boutique typography if it weakens clinical trust
- design decisions that reduce clarity around evaluation, pricing, or support
- excessive whitespace that makes important medical-adjacent information feel buried

Skinfluence still needs to communicate:
- provider review
- realistic expectations
- treatment seriousness
- support routing clarity

## Immediate Visual Priorities

If we use Portland as the benchmark, the most important Skinfluence upgrades are:

1. real photography in key slots
- homepage hero
- homepage side-by-side block
- start-evaluation hero
- support hero

2. stronger section variation
- homepage should alternate between light, banded, and darker moments

3. cleaner page-specific hero identities
- pricing, support, FAQ, and evaluation should not all feel like the same page

4. better content-to-image balance
- some sections should feel more visual
- others should feel more editorial and text-led

## Success Criteria

Skinfluence should feel:
- more like an editorial anti-aging brand
- less like a custom theme that still shows its scaffolding
- more visually intentional page to page
- more premium through imagery and pacing, not through decorative excess

If a future design pass makes the storefront feel calmer, more distinct page by page, and more visually led without losing clarity, then the Portland reference is being used correctly.
