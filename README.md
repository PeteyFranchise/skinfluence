# Skinfluence

Skinfluence is a premium clinical skincare brand concept centered on provider-reviewed, tretinoin-led anti-aging care.

This repository is the project workspace for strategy, architecture, storefront planning, UX/content systems, and future implementation artifacts.

## Repository structure

- `docs/rules/`
  Project operating rules and workflow references used by future contributors and AI tools.
- project root
  Current planning artifacts, visual references, and working deliverables already created for the project.

## Workflow rules

The master workflow rules for this repo live at:

- [docs/rules/MASTER-WORKFLOW-RULES.md](docs/rules/MASTER-WORKFLOW-RULES.md)

These rules should be treated as the baseline operating guidance for future project work unless explicitly overridden.
